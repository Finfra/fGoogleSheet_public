# fGoogleSheet

> **가장 빠른 Google Sheets Mac 메뉴바 앱.**

macOS 메뉴바에서 Google Sheets에 초고속으로 접근하는 앱입니다. Google 계정을 연결하고 메뉴바에서 직접 스프레드시트를 관리하세요.

## 주요 기능

- **초고속 접근** - 메뉴바에서 Google Sheets에 직접 접근
- **원클릭 동기화** - Google 계정과 실시간 동기화
- **워크플로우 유지** - 앱 전환 없이 스프레드시트 확인 및 편집
- **맞춤 환경** - 단축키 및 실행 설정 커스터마이징
- **시트 관리** - 즐겨찾기 및 숨김 기능으로 빠른 접근
- **안전한 연동** - Google OAuth 기반 보안 인증

## 시스템 요구사항

- macOS 14.0 이상
