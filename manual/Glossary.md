# 용어 사전 (Glossary)

이 문서는 fGoogleSheet 프로젝트 내의 주요 용어와 다국어 번역 기조를 정리한 문서입니다. 개발 및 매뉴얼 작성, UI 수정 시 이 용어를 기준으로 합니다.

| 설명(개발용)                      | English           | Korean            | Japanese               | German                 | Spanish                    | French                       |
| :-------------------------------- | :---------------- | :---------------- | :--------------------- | :--------------------- | :------------------------- | :--------------------------- |
| 핵심 입력 데이터(질문/항목명 등)  | Key               | 키                | キー                   | Schlüssel              | Clave                      | Clé                          |
| 대응 입력 데이터(답변/내용 등)    | Value             | 값 (답변)         | 値                     | Wert                   | Valor                      | Valeur                       |
| Google Sheets 접근을 위한 인증 키 | API Key           | API 키            | APIキー                | API-Schlüssel          | Clave de API               | Clé API                      |
| 대상이 되는 스프레드시트의 ID     | Document ID       | 문서 ID           | ドキュメントID         | Dokument-ID            | ID de documento            | ID du document               |
| 데이터를 입력할 시트의 활성 탭    | Sheet Tab         | 시트 탭           | シートタブ             | Tabellenblatt          | Pestaña de hoja            | Onglet de feuille            |
| 빈 행을 찾아 데이터를 추가함      | Append            | 이어쓰기 (추가)   | 追加                   | Anhängen               | Añadir                     | Ajouter                      |
| 답변을 찾지 못한 질문 항목        | Unanswered        | 미답변 질문       | 未回答の質問           | Unbeantwortet          | Sin respuesta              | Sans réponse                 |
| 앱의 동작을 제어하는 옵션         | Settings          | 설정              | 設定                   | Einstellungen          | Configuración              | Paramètres                   |
| 내장 HTTP API 서버                | REST API Server   | REST API 서버     | REST APIサーバー       | REST-API-Server        | Servidor REST API          | Serveur REST API             |
| IP 범위 허용 표기법               | CIDR              | CIDR              | CIDR                   | CIDR                   | CIDR                       | CIDR                         |
| 서버 동작 확인 요청               | Health Check      | 헬스 체크         | ヘルスチェック         | Gesundheitscheck       | Verificación de estado     | Vérification de santé        |
| 교차 출처 리소스 공유             | CORS              | CORS              | CORS                   | CORS                   | CORS                       | CORS                         |
